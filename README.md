# 🧮 JavaFX Calculator for Desktop

A simple yet elegant desktop calculator application built using **Java** and **JavaFX**.  
This project was developed as a hands-on practice to learn JavaFX, FXML-based UI design, and event-driven programming concepts.

---

## 📌 Project Overview

This calculator performs basic arithmetic operations through a clean and user-friendly interface.  
The application demonstrates how to separate UI design from logic using FXML and apply custom styling using CSS.

It is designed to be simple, responsive, and easy to use.

## 🚀 Technologies Used

- JavaFX

## ✨ Features

- ✅ Addition, Subtraction, Multiplication, and Division
- ✅ Clean and intuitive user interface
- ✅ Event-driven button handling
- ✅ Error handling for invalid inputs
- ✅ Structured and maintainable code

## 🛠️ Concepts Practiced
